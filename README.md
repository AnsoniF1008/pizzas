# Pizza Trainer

App de estudio para aprender a armar las 14 pizzas de la Sección 1: repaso, tarjetas de memoria y examen.

## Desarrollo local

```bash
npm install
npm run dev
```

## Compilar

```bash
npm run build
```

## Desplegar a Firebase Hosting

```bash
npx -y firebase-tools@latest deploy --only hosting
```
